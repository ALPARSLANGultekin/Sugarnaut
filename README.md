# Sugarnaut - Mobile Puzzle Game 🎮🍬

Sugarnaut is an end-to-end mobile puzzle game developed using the Unity engine. While the original source code was lost due to a hardware reset, the fully completed, playable, and compiled version (APK) is preserved and showcased here.

## 🚀 Project Overview
This project demonstrates mobile game mechanics, level mapping, and UI development. The development process involved rigorous debugging and fine-tuning of the core game loop.

## ✨ Key Features
* **Precise Game Loop Mechanics:** Finely tuned win/loss state timing that accurately waits for the final piece to fall and resolve before declaring the end of a level.
* **Health & Progression System:** Synchronized health deduction upon level failure and a progressively challenging level map.
* **Engaging UI/UX:** Custom puzzle-solving interactions and smooth gameplay mechanics.

## 📥 Download and Play (Android)
You can download and play the final build of the game on your Android device:
1. Go to the **Releases** section on the right side of this repository.
2. Download the `Sugarnaut.apk` file.
3. Open the APK file on your phone to install and play.
